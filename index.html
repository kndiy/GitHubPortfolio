3. Download From AWS S3 by Time Class

This small class stems from my need to query S3 files.

This class can be initiated in to an Object to help downloading files (byte arrays actually) from AWS S3 using time String (yyyyMMddHHmmss) or last/ first modified. Depending on the method called, the following actions are done:

  • Create a TreeMap of all object keys inside a specific S3 folder,
  • Either choose FILE_LOCATION: FIRST/LAST or specify your choice of time in String format: yyyyMMddHHmmss to initiate the download,
  • The TreeMap is saved as an instant variable so that we can re-query without parsing all object summaries again.
